<br><br><h2 style="color:#0070c0; border-bottom:2px solid #0070c0; padding-bottom:5px; margin-top:30px;">"More": The Secrets of Tool Holders and Collision Detection</h2><br><p>Listen up, this 'Check Tool and Clamping Device' feature is very useful. You need to understand that only when the system knows the shape and dimensions of your tool holder can it help you perform collision checks. If you haven't created the tool holder, the system is flying blind and can't check anything.</p>
<br><p>So, to make good use of this function, the first step is to create the tool holder. Generally, we can set up the tool holder at the same time as we define the clamping device.</p>
<br><br><h3 style="color:#3399ff; border-left:5px solid #ff9900; padding-left:10px; margin:25px 0;">How to Set Tool Holder Parameters</h3><br><p>Additionally, you can find the clamping device settings under 'Tool's' 'Edit Display'. As long as you know the tool holder parameters, you can set them here. This part is relatively simple; you can figure it out yourselves with a bit of exploration.</p>
<br><p>Typically, Siemens NX's (formerly UG NX) simulation software itself comes with some tool holder settings, but for those of us doing practical work, we need to have a clear understanding.</p>

<br><br><h3 style="color:#3399ff; border-left:5px solid #ff9900; padding-left:10px; margin:25px 0;">The Practical Meaning of Safe Distance</h3><br><p>Alright, let's reopen it now. In the 'Operation Navigator', the spatial range has pretty much been covered. Let's look inside 'More', there's a parameter for 'Clamping Device'.</p>
<br><p>See it? The default is <strong style="color:#e46c0a;">3mm</strong>. This small arrow points to the safe distance from the edge of the tool holder to the workpiece. This means that the closest point of our tool holder to the workpiece must maintain a minimum clearance of <strong style="color:#e46c0a;">3mm</strong>. This is to prevent the tool holder from colliding with the workpiece during machining, which could lead to accidents or scrapped parts.</p>
<br><p>Let's change it, for example, input <strong style="color:#e46c0a;">10mm</strong>. Isn't it obviously much larger? From here to here, it now maintains a safe distance of <strong style="color:#e46c0a;">10mm</strong>. Master Wang reminds you, this distance is critical for safety!</p>
<br><br><h3 style="color:#3399ff; border-left:5px solid #ff9900; padding-left:10px; margin:25px 0;">The Nuances of Collision Check and Pitfall Avoidance</h3><br><p>You all now understand the meaning of 'More' here; it's about providing a <strong style="color:#e46c0a;">safe distance</strong> for the tool holder. If 'Collision Check' is not ticked, no matter what number you enter here, the system will ignore it! Because you haven't activated the collision detection function, it won't read your set safe distance. It's like telling someone you're afraid of heights, and then going skydiving ‚Äì who's to blame then?</p>
<br><p>So, for this safe distance to take effect, <strong style="color:#e46c0a;">you must tick that 'Collision Check' box!</strong> Otherwise, no matter how many numbers you input, it will treat you as air. These are lessons learned the hard way; don't make such basic mistakes.</p>

<br><p>Usually, when programming 3-axis toolpaths, we don't need these functions because the tool holder is generally far from the workpiece. But for 5-axis or complex cavity machining, you really need to make good use of them.</p>
<br><br><h2 style="color:#0070c0; border-bottom:2px solid #0070c0; padding-bottom:5px; margin-top:30px;">"Connection": Optimizing Toolpaths, Boosting Efficiency</h2><br><p>Having finished 'More', let's talk about 'Connection'. This function directly relates to your machining efficiency and machine tool lifespan, so listen closely!</p>
<br><p>First, let's change the tool. Earlier, we used a 10mm diameter tool and set its tool holder. Now, if we switch to a 12mm diameter tool, you'll find the tool holder is gone. Why? Because we only set the clamping device for the 10mm tool; the 12mm tool hasn't been set yet. NX isn't that smart; not all tools share one clamping device setting.</p>
<br><p>We'll now demonstrate using a 'Face Milling' (ÂÖâÂ∫ïÂ£Å) machining method. This is a very common machining strategy.</p>
<br><br><h3 style="color:#3399ff; border-left:5px solid #ff9900; padding-left:10px; margin:25px 0;">Region Sorting: Optimize vs. Standard</h3><br><p>Let's look at the 'Region Sorting' option under 'Connection'. There are two types: 'Optimize' and 'Standard'.</p>

<div align="center"><img src="https://raw.githubusercontent.com/cncabc-sketch/Siemens-NX-CAM-Technical-Guides/main/Articles/IMG/%E7%AC%AC15%E8%8A%82%E8%AF%BE%20%20%E4%B8%89%E8%BD%B4/more-connection-03.webp" style="max-width:100%; border-radius:10px;"></div>

<br><p>For example, if we're machining several regions. If you choose 'Optimize', the system will <strong style="color:#e46c0a;">automatically</strong> plan the most efficient path for you. It won't rigidly follow a sequence, but rather, based on the actual situation, it will choose the closest, smoothest route. For instance, it might machine this region first, then that one, then jump to another ‚Äì all calculated by the computer. This greatly reduces air cut time, and saving money is making money!</p>
<br><p>If you choose 'Standard', it's not as intelligent. It might machine this first, then run to a distant one, then jump back, darting all over the place. This will create many unnecessary air cuts, wasting time and increasing machine wear, which is not worth it. So, Master Wang tells you, in general, <strong style="color:#e46c0a;">just choose 'Optimize'</strong>, don't mess around with those infrequently used options.</p>
<br><br><h3 style="color:#3399ff; border-left:5px solid #ff9900; padding-left:10px; margin:25px 0;">Cross-Empty-Space Motion Type: Cut, Follow, Move Tool</h3><br><p>Next is 'Cross-Empty-Space Motion Type', which includes 'Cut', 'Follow', and 'Move Tool'.</p>
<br><p>In 'Cut' mode, if there are voids or discontinuities in the middle of the machining area, the toolpath will go straight through, as if it didn't see them. This might leave some paths in the air, but for some simple shapes, it's also a method.</p>
<br><p>Let's switch to a new operation to demonstrate, otherwise, changing too many parameters can lead to problems.</p>

<div align="center"><img src="https://raw.githubusercontent.com/cncabc-sketch/Siemens-NX-CAM-Technical-Guides/main/Articles/IMG/%E7%AC%AC15%E8%8A%82%E8%AF%BE%20%20%E4%B8%89%E8%BD%B4/more-connection-04.webp" style="max-width:100%; border-radius:10px;"></div>

<br><p>Okay, now let's select a bottom face and use a smaller tool, like a D3. Then, change 'Cut' to 'Follow' and generate it.</p>
<br><p>It's clear now, isn't it? In 'Follow' mode, the tool will stick close to the boundary, instead of cutting straight through like 'Cut'. This avoids tool movement in areas that shouldn't be machined, saving some time.</p>
<br><p>Let's try 'Move Tool'. 'Move Tool' means that when the tool moves from one cutting region to another non-connected cutting region, it will lift up and travel in a <strong style="color:#e46c0a;">rapid move (G00)</strong>, with a blue line indicating this intermediate section. See this blue line? It means <strong style="color:#e46c0a;">rapid move</strong>. This can greatly shorten the idle travel time. For example, it moves to this position, finishes cutting, and needs to go to another position; it will lift the tool and rapid move there directly, very quickly. Only when it reaches the new cutting point does it resume normal cutting speed. This can save a lot of machining time, especially for complex parts.</p>
<br><p>So, 'Cut', 'Follow', and 'Move Tool' are all quite commonly used, just choose flexibly according to the actual situation.</p>
<br><br><h2 style="color:#0070c0; border-bottom:2px solid #0070c0; padding-bottom:5px; margin-top:30px;">Finishing Pass Toolpath: 'Beautifying' the Workpiece</h2><br><p>Finally, let's look at the 'Add Finishing Pass Toolpath' function.</p>
<br><p>For example, after machining a region, if you want its edges to be smoother and dimensions more accurate, you can use this. It will add an extra pass along the already machined path, which is equivalent to a 'finish cut' (ÂÖâÂàÄ), to clean up the remaining material on the surface. It's like 'beautifying' the workpiece.</p>
<br><p>Here you can also set the 'Number of Passes'. For instance, if you want to make a few passes along the edge, just input that number. It might be used less often normally, but it can be very useful at critical moments.</p>
