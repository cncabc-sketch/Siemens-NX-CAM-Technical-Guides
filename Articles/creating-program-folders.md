It represents the highest level of operation, and all programs should be housed under NC, not nested within other lettered folders. If placed under A, then A becomes its parent, and if A is deleted, everything below it is gone too.</li><br><li>Click "OK" to complete the creation.</li><br></ol><br><h2 style="color:#0070c0; border-bottom:2px solid #0070c0; padding-bottom:5px; margin-top:30px;">Naming Rules and File Duplication</h2><br><p>When creating program folders, there's a common pitfall: <strong style="color:#e46c0a; font-weight:bold;">duplicate naming</strong>.</p><br><h3 style="color:#3399ff; border-left:5px solid #ff9900; padding-left:10px; margin:25px 0;">Why Does "-1" Automatically Get Added?</h3><br><p>When you try to create a program folder with the same name as an existing one (e.g., if "F" already exists and you create another "F"), NX will automatically append a <strong style="color:#e46c0a;">-1</strong> suffix to the newly created folder, making it "F-1," or even "F01-1."</p><br><p>This is a mechanism within the software to prevent file conflicts and maintain uniqueness. Instead, we right-click and choose "Insert Operation." Next time, I'll start with the DB template and, following my teaching sequence, explain all the contents within the templates clearly.</p><br><p>There are many tutorials out there, but as long as you follow Master Wang's approach, I guarantee you'll be able to get hands-on work done after learning!</p><br><h3 style="color:#3399ff; border-left:5px solid #ff9900; padding-left:10px; margin:25px 0;">Summary: Pitfall Guide</h3><br><ul><br><li><strong style="color:#e46c0a; font-weight:bold;">Program Location must be NC</strong>: When creating program folders, their location should always be below <strong style="color:#e46c0a;">NC</strong> to ensure correct hierarchy.</li><br><li><strong style="color:#e46c0a; font-weight:bold;">Understand the Auto "-1" Mechanism</strong>: If you see names automatically suffixed with <strong style="color:#e46c0a;">-1</strong>, it's because a file with the same name already exists. Either delete the old one or accept the system's automatic numbering.</li><br><li><strong style="color:#e46c0a; font-weight:bold;">Drag-and-Drop Organization Requires Caution</strong>: When dragging folders, be sure to clearly observe the blue highlight indicating the cursor's position.
