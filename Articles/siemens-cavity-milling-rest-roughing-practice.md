<p>Hello everyone, I'm Master Wang. Last time, we pretty much covered roughing; today, we'll continue with <strong style="color:#e46c0a; font-weight:bold;">rest roughing</strong>. Don't underestimate it – when used well, this technique can save you a lot of finishing time and effectively improve the part's surface quality.</p>
<br><br><h2 style="color:#0070c0; border-bottom:2px solid #0070c0; padding-bottom:5px; margin-top:30px;">Rest Roughing Operation: Siemens NX Command Selection and Initial Setup</h2><br><br><p>Listen closely: for rest roughing in NX, we'll directly choose the "Cavity Milling Rest Roughing" command. Simply put, it's a branch of cavity milling, but NX engineers have pre-set some parameters for you, making it more suitable for corner cleanup and processing material left from the previous roughing pass. If you were to use it for initial roughing, it's not impossible, but you'd have to change a bunch of parameters—why create trouble for yourself? It's time-consuming, laborious, and not worth it!</p>
<br><ul><br>    <li><strong style="color:#e46c0a; font-weight:bold;">Operation Entry:</strong> Directly select <strong style="color:#e46c0a;">"Cavity Milling Rest Roughing"</strong>.</li><br>    <li><strong style="color:#e46c0a; font-weight:bold;">Stock and Part:</strong> This section is the same as the initial roughing; just click OK, no need to change. The system will default to your previous settings.</li><br>    <li><strong style="color:#e46c0a; font-weight:bold;">Tool Selection:</strong> Rest roughing, as the name suggests, is used to clear corners unreachable by the large tool during the previous roughing pass. Therefore, you need to select a tool with a smaller diameter, typically a ball end mill or bull nose end mill, and a smaller corner radius (R value) than the roughing tool. For example, if you used a D25R3 for roughing, you might consider a D12R1 for rest roughing.</li><br>    <li><strong style="color:#e46c0a; font-weight:bold;">Cutting Parameters:</strong><br>        <ul><br>            <li><strong style="color:#e46c0a; font-weight:bold;">Cut Pattern:</strong> Generally, use <strong style="color:#e46c0a;">"Contour Part"</strong>; this is the most common method.</li><br>            <li><strong style="color:#e46c0a; font-weight:bold;">Depth of Cut (DOC):</strong> This parameter is crucial. We typically set it to <strong style="color:#e46c0a;">around 0.5mm</strong>, depending on material hardness and tool strength. Don't make it too large; corner cleanup is not about aggressive roughing, it needs to be stable.</li><br>            <li><strong style="color:#e46c0a; font-weight:bold;">Cutting Depth:</strong> If there are no special requirements, such as not wanting to mill a deep hole all the way through, or if a specific area has depth restrictions, then usually keep the default. Don't change it arbitrarily, or the toolpath will be messy.</li><br>        </ul><br>    </li><br></ul><br><div style="margin:35px 0; text-align:center;"><img src="IMG/第48节课  三轴/siemens-cavity-milling-rest-roughing-practice-01.webp" style="width:100%; border-radius:15px; box-shadow:0 10px 30px rgba(0,0,0,0.15);" /></div><br><br><h2 style="color:#0070c0; border-bottom:2px solid #0070c0; padding-bottom:5px; margin-top:30px;">Core Secrets: Correct Usage of In-Process Workpiece and Reference Tool</h2><br><br><p>Alright, now let's talk about the two most critical points for rest roughing, where many novices often stumble. Listen up!</p>

<div align="center"><img src="https://raw.githubusercontent.com/cncabc-sketch/Siemens-NX-CAM-Technical-Guides/main/Articles/IMG/%E7%AC%AC48%E8%8A%82%E8%AF%BE%20%20%E4%B8%89%E8%BD%B4/siemens-cavity-milling-rest-roughing-practice-01.webp" style="max-width:100%; border-radius:10px;"></div>

<br><div style="margin:35px 0; text-align:center;"><img src="IMG/第48节课  三轴/siemens-cavity-milling-rest-roughing-practice-02.webp" style="width:100%; border-radius:15px; box-shadow:0 10px 30px rgba(0,0,0,0.15);" /></div><br><br><h3 style="color:#3399ff; border-left:5px solid #ff9900; padding-left:10px; margin:25px 0;">"In-Process Workpiece": The "Eyes" of Rest Roughing</h3><br><p>Why is it that even though you've set up rest roughing, the toolpath isn't calculated? Or it calculates a bunch of redundant toolpaths? The problem lies here:</p>
<br><ul><br>    <li><strong style="color:#e46c0a;">Core Setting:</strong> In the rest roughing parameter settings, find the <strong style="color:#e46c0a;">"Geometry"</strong> tab, and set <strong style="color:#e46c0a;">"In-Process Workpiece"</strong> to <strong style="color:#e46c0a;">"5"</strong>! Remember, it's the number "5"!</li><br>    <li><strong style="color:#e46c0a; font-weight:bold;">Principle:</strong> Setting it to "5" tells the system: "Hey, I've already machined this with a previous tool; now, show me where the unmachined residual material is, and only machine those areas!" If you don't set it to "5", the system won't know what you did before; it will treat it as initial roughing, and naturally, it will get confused, either failing to calculate a toolpath or generating a bunch of useless ones.</li><br></ul><br><br><h3 style="color:#3399ff; border-left:5px solid #ff9900; padding-left:10px; margin:25px 0;">"Reference Tool": The "Compass" of Rest Roughing</h3><br><p>Just setting "In-Process Workpiece" isn't enough; you also need to point it in the right direction.</p>
<br><ul><br>    <li><strong style="color:#e46c0a;">Function:</strong> The "Reference Tool" tells the system which tool was used previously and its size. Based on the shape and size of this reference tool, combined with the "In-Process Workpiece" instruction, the system can accurately calculate where residual material remains and needs to be cleared by the current smaller tool.</li><br>    <li><strong style="color:#e46c0a; font-weight:bold;">Selection Technique:</strong><br>        <ul><br>            <li><strong style="color:#e46c0a; font-weight:bold;">First Rest Roughing Pass:</strong> Select the large tool you used for the previous roughing pass. For instance, if you used a D25R3 for roughing, then for the first rest roughing pass, select <strong style="color:#e46c0a;">D25R3</strong> as the reference tool.</li><br>            <li><strong style="color:#e46c0a; font-weight:bold;">Multiple Rest Roughing Passes:</strong> If you need to perform multi-level rest roughing (e.g., D25R3 → D12R1 → D8R1), then each level of rest roughing must reference <strong style="color:#e46c0a;">the machining tool from its preceding level</strong>. For example, when using a D8R1 for rest roughing, the reference tool should be <strong style="color:#e46c0a;">D12R1</strong>.</li><br>            <li><strong style="color:#e46c0a; font-weight:bold;">Create New Tool:</strong> If the corresponding reference tool isn't in your library, simply create a new one and ensure the parameters are set correctly. The important thing is that the parameters accurately reflect the dimensions of the previous tool.</li><br>        </ul><br>    </li><br>    <li><strong style="color:#e46c0a; font-weight:bold;">Engagement Strategy:</strong> For parameters like <strong style="color:#e46c0a;">"Plunge Engage"</strong>, I typically set it to <strong style="color:#e46c0a;">1mm</strong> to ensure stable engagement and prevent excessive cutting impact.</li><br></ul><br><div style="margin:35px 0; text-align:center;"><img src="IMG/第48节课  三轴/siemens-cavity-milling-rest-roughing-practice-03.webp" style="width:100%; border-radius:15px; box-shadow:0 10px 30px rgba(0,0,0,0.15);" /></div><br><br><h2 style="color:#0070c0; border-bottom:2px solid #0070c0; padding-bottom:5px; margin-top:30px;">Striving for Excellence: Toolpath Optimization and Practical Experience</h2><br><br><h3 style="color:#3399ff; border-left:5px solid #ff9900; padding-left:10px; margin:25px 0;">The Secret of "Minimum Residual Material"</h3><br><p>This parameter is also a critical one; often, an unclean toolpath or mysterious small paths are related to it.</p>

<div align="center"><img src="https://raw.githubusercontent.com/cncabc-sketch/Siemens-NX-CAM-Technical-Guides/main/Articles/IMG/%E7%AC%AC48%E8%8A%82%E8%AF%BE%20%20%E4%B8%89%E8%BD%B4/siemens-cavity-milling-rest-roughing-practice-02.webp" style="max-width:100%; border-radius:10px;"></div>

<br><ul><br>    <li><strong style="color:#e46c0a; font-weight:bold;">Definition:</strong> <strong style="color:#e46c0a;">"Minimum Residual Material"</strong> means that if the thickness of the material remaining in a certain area after the previous machining is smaller than the value you set, then the current tool will not machine it.</li><br>    <li><strong style="color:#e46c0a; font-weight:bold;">Application:</strong> For example, if you set it to 0.2mm, then areas with only 0.1mm or less material remaining will be considered by the system as "not necessary to cut, leave it for finishing or the next smaller tool," and thus ignored.</li><br>    <li><strong style="color:#e46c0a; font-weight:bold;">Master Wang's Experience:</strong><br>        <ul><br>            <li>This value should typically be set to <strong style="color:#e46c0a;">half of the previous tool's "Depth of Cut" (DOC), or slightly less than the corner radius of your current machining tool.</strong></li><br>            <li>For example, if the previous tool's DOC was 1mm, then this value can be set to 0.5mm, or even slightly smaller like 0.4mm or 0.3mm.</li><br>            <li>Setting this value appropriately can effectively prevent the tool from cutting tiny scraps, which wastes time, causes tool wear, and reduces efficiency.</li><br>        </ul><br>    </li><br></ul><br><div style="margin:35px 0; text-align:center;"><img src="IMG/第48节课  三轴/siemens-cavity-milling-rest-roughing-practice-04.webp" style="width:100%; border-radius:15px; box-shadow:0 10px 30px rgba(0,0,0,0.15);" /></div><br><br><h3 style="color:#3399ff; border-left:5px solid #ff9900; padding-left:10px; margin:25px 0;">"Trim Boundaries": A Great Helper for Streamlining Toolpaths</h3><br><p>Rest roughing toolpaths can sometimes be a bit redundant, especially in less critical areas.</p>
<br><ul><br>    <li><strong style="color:#e46c0a; font-weight:bold;">Function:</strong> Through <strong style="color:#e46c0a;">"Trim Boundaries"</strong>, you can manually specify a point or area to make the tool avoid these places and stop generating toolpaths there.</li><br>    <li><strong style="color:#e46c0a; font-weight:bold;">Purpose:</strong> This helps you optimize the toolpath, making it more streamlined and efficient. Sometimes, the residual material in certain areas doesn't need to be cleared again, or you want to handle it in another way, so you can trim it.</li><br></ul><br><br><h3 style="color:#3399ff; border-left:5px solid #ff9900; padding-left:10px; margin:25px 0;">Multi-Level Rest Roughing: Layer by Layer, Step by Step</h3><br><p>For complex cavities or high-precision requirements, a single rest roughing pass is often insufficient. We can proceed layer by layer, like peeling an onion, to go deeper.</p>
<br><ul><br>    <li><strong style="color:#e46c0a; font-weight:bold;">Approach:</strong> You can duplicate the current rest roughing operation, then switch to a smaller tool, and simultaneously update the "Reference Tool" to the machining tool from the previous level.</li><br>    <li><strong style="color:#e46c0a; font-weight:bold;">Example:</strong><br>        <ol><br>            <li>D25R3 roughing.</li><br>            <li>D12R1 rest roughing, referencing D25R3.</li><br>            <li>D8R1 rest roughing, referencing D12R1.</li><br>        </ol><br>    This is called a <strong style="color:#e46c0a;">"Corner Cleanup Sequence"</strong>, which ensures every corner is thoroughly cleared, laying a solid foundation for the final finishing pass.</li><br></ul><br><br><h3 style="color:#3399ff; border-left:5px solid #ff9900; padding-left:10px; margin:25px 0;">Allowance Control: Making Toolpaths Smoother</h3><br><p>Here's another small detail that can make your rest roughing toolpaths "smarter."</p>

<div align="center"><img src="https://raw.githubusercontent.com/cncabc-sketch/Siemens-NX-CAM-Technical-Guides/main/Articles/IMG/%E7%AC%AC48%E8%8A%82%E8%AF%BE%20%20%E4%B8%89%E8%BD%B4/siemens-cavity-milling-rest-roughing-practice-03.webp" style="max-width:100%; border-radius:10px;"></div>

<br><ul><br>    <li><strong style="color:#e46c0a; font-weight:bold;">Master Wang's Experience:</strong> During rest roughing, the allowance left should ideally be <strong style="color:#e46c0a;">slightly smaller than during initial roughing</strong>. For example, if roughing leaves 0.35mm, rest roughing can leave 0.25mm or 0.2mm.</li><br>    <li><strong style="color:#e46c0a; font-weight:bold;">Benefit:</strong> Doing so prevents the tool from repeatedly cutting areas that have already been cleared, making the toolpath smoother, reducing air cutting, and improving efficiency.</li><br></ul><br><br><h3 style="color:#3399ff; border-left:5px solid #ff9900; padding-left:10px; margin:25px 0;">Machining Simulation and Verification</h3><br><p>NX's simulation function is very powerful, but don't just watch it run through once and call it a day.</p>
<br><ul><br>    <li><strong style="color:#e46c0a; font-weight:bold;">Key Point:</strong> During simulation, pay special attention to the tool's movement in complex areas like corners and deep cavities. Compare before and after rest roughing to see if the residual material in these areas has been effectively removed.</li><br>    <li><strong style="color:#e46c0a; font-weight:bold;">Details:</strong> Some subtle residual material might only be discovered by zooming in and carefully observing the simulation. Don't rely solely on software simulation; during actual machining, you also need to observe cutting sparks and listen to cutting sounds—those are the most authentic feedback.</li><br></ul><br><br><h2 style="color:#0070c0; border-bottom:2px solid #0070c0; padding-bottom:5px; margin-top:30px;">Master Wang's Summary: Pitfall Avoidance Guide</h2><br><p>In our line of work, theoretical knowledge alone isn't enough; practical experience is key. For today's rest roughing, just remember these points to ensure you take the shortest path to success:</p>
<br><ol><br>    <li><strong style="color:#e46c0a;">In-Process Workpiece:</strong> Always <strong style="color:#e46c0a;">"5"</strong>! This is the soul of rest roughing; without it, everything else is moot.</li><br>    <li><strong style="color:#e46c0a;">Reference Tool:</strong> Absolutely select <strong style="color:#e46c0a;">the tool actually used in the preceding machining level</strong>. This is the eyes of rest roughing, telling the system where material still remains.</li><br>    <li><strong style="color:#e46c0a;">Minimum Residual Material:</strong> Set it appropriately, usually <strong style="color:#e46c0a;">smaller than the previous tool's depth of cut and the current tool's radius</strong>, to avoid meaningless small cuts, protect the tool, and improve efficiency.</li><br>    <li><strong style="color:#e46c0a;">Decreasing Allowance:</strong> The allowance for rest roughing should be <strong style="color:#e46c0a;">slightly smaller than the initial roughing pass</strong>, making the toolpath smoother and avoiding redundant machining.</li><br>    <li><strong style="color:#e46c0a;">Toolpath Optimization:</strong> Use <strong style="color:#e46c0a;">"Trim Boundaries"</strong> to remove redundant toolpaths, making the program more streamlined.</li><br>    <li><strong style="color:#e46c0a;">Multi-Level Corner Cleanup:</strong> For complex parts or those requiring high precision, consider using <strong style="color:#e46c0a;">multi-level rest roughing</strong>, progressing with tools of different diameters layer by layer to thoroughly remove all residual material.</li><br></ol><br><p>These are experiences I've accumulated over 15 years of hard work on the front lines; you might not find them in textbooks, but they'll definitely be useful to you! Go back, digest this, practice more on the machine, and if you don't understand, come ask me again.</p>
